# Engine Web

While the extension is our main focus, this package allows the engine to live in a standard web page. It’s a slimmed-down, browser-compatible version of our logic.

### What it does

- **Portability:** It bundles the core engine so it can be used in any regular web application without needing a browser extension.
- **Simple Setup:** Provides a straightforward way to initialize workflows directly from JavaScript.
- **Global Access:** Makes the engine available through a simple interface, allowing web developers to trigger actions or processes easily.
